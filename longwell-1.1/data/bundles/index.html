Data Bundles

This is the collection of data bundles produced by the SIMILE project.

  • art-public - this data bundle is about digital libraries of artistic material and it's composed by datasets that are publicly redistributable.
  • art-private - this data bundle is about digital libraries of artistic material but it's composed by datasets that are not all publicly redistributable
    • this bundle includes datasets that are not available to the public
  • tr - this data bundle is about technical reports
  • people - this data bundle is about people, their information and their relationships
  • glossary - this data bundle is about glossaries of technical terms
  • bibliography - this data bundle is about bibliographies
  • code - this data bundle is about description and dependencies between source code files and parts
  • nci - this data bundle is about the NCI's thesaurus of cancer terms
